# Employee Management System

This is a Java Spring Boot based Employee Management System with role-based authentication for Admin and User.

## Features
- Admin can manage employees (Add, Edit, Delete)
- User can view employee details
- Secure login with Spring Security
- Role-based access
- Responsive UI using HTML + CSS + Bootstrap

## Technology Stack
- **Backend:** Java, Spring Boot, Spring Security
- **Frontend:** Thymeleaf, HTML, CSS, Bootstrap
- **Database:** MySQL
